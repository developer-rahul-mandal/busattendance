# বাস উপস্থিতি ব্যবস্থাপনা সিস্টেম

এটি একটি সুপার অ্যাডমিন লগইন সিস্টেম যা বাস উপস্থিতি ব্যবস্থাপনার জন্য তৈরি করা হয়েছে।

## বৈশিষ্ট্য

- ✅ সুপার অ্যাডমিন লগইন সিস্টেম
- ✅ ডেটাবেস ইন্টিগ্রেশন
- ✅ সেশন ম্যানেজমেন্ট
- ✅ সিকিউর লগইন (পাসওয়ার্ড হ্যাশিং)
- ✅ রেসপন্সিভ ডিজাইন
- ✅ বাংলা ভাষা সমর্থন

## ইনস্টলেশন

### প্রয়োজনীয় সফটওয়্যার
- XAMPP (Apache, MySQL, PHP)
- PHP 7.4 বা তার উপরের সংস্করণ
- MySQL 5.7 বা তার উপরের সংস্করণ

### সেটআপ নির্দেশনা

1. **প্রজেক্ট ফোল্ডার সেটআপ**
   ```
   C:\xampp\htdocs\busattendance\
   ```

2. **ডেটাবেস তৈরি করুন**
   - XAMPP Control Panel থেকে MySQL চালু করুন
   - phpMyAdmin খুলুন (http://localhost/phpmyadmin)
   - নতুন ডেটাবেস তৈরি করুন: `bus_attendance`

3. **ডেটাবেস সেটআপ চালান**
   ```
   http://localhost/busattendance/setup_database.php
   ```

4. **লগইন করুন**
   ```
   http://localhost/busattendance/login.php
   ```

## ডিফল্ট লগইন তথ্য

- **ইউজারনেম:** admin
- **পাসওয়ার্ড:** admin123

## ফাইল স্ট্রাকচার

```
busattendance/
├── config/
│   └── database.php          # ডেটাবেস কনফিগারেশন
├── login.php                 # লগইন পেজ
├── process_login.php         # লগইন প্রসেসিং
├── dashboard.php             # সুপার অ্যাডমিন ড্যাশবোর্ড
├── logout.php                # লগআউট ফাংশন
├── setup_database.php        # ডেটাবেস সেটআপ স্ক্রিপ্ট
├── index.php                 # হোম পেজ (লগইনে রিডাইরেক্ট)
└── README.md                 # এই ফাইল
```

## সিকিউরিটি বৈশিষ্ট্য

- পাসওয়ার্ড হ্যাশিং (PHP password_hash())
- সেশন ম্যানেজমেন্ট
- SQL ইনজেকশন সুরক্ষা (PDO Prepared Statements)
- সেশন টাইমআউট (২৪ ঘন্টা)
- CSRF সুরক্ষা

## ব্যবহার

1. প্রথমে `setup_database.php` চালান
2. `login.php` থেকে লগইন করুন
3. সফল লগইনের পর `dashboard.php` এ রিডাইরেক্ট হবে
4. ড্যাশবোর্ড থেকে সমস্ত কার্যক্রম পরিচালনা করুন
